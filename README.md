# Password Security Analyzer 

A Python-based cybersecurity project focused on password security analysis.
This tool evaluates password strength, calculates entropy, demonstrates
dictionary attacks on weak passwords, and generates strong passwords.

## Features
- Rule-based password strength checking
- Entropy calculation (security math)
- Dictionary attack demonstration (educational)
- Strong password generator

## Technologies Used
- Python
- Regex
- Hashlib
- Math module

## Ethical Disclaimer
This project is strictly for educational and defensive security purposes.
It must not be used for unauthorized access or illegal activities.


## Project Structure
- password_strength.py – Rule-based strength analysis
- entropy.py – Entropy calculation logic
- password_crack_demo.py – Dictionary attack demonstration
- password_generator.py – Secure password generation
- wordlist.txt – Sample dictionary (educational)
